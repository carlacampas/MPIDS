------------------------ Manual de uso ------------------------
Para utilizar los diferentes programas simplemente hay que abrir la terminal y usar el comando 
                ./script.sh

Una vez ejecutado el script, se pedirá cuál de los archivos se quiere ejecutar (greedy, local_search o metaheuristica) y el número de veces que se quiere ejecutar el programa. Se compilará automáticamente y posteriormente veremos escritos los diferentes archivos y los resultados obtenidos.
